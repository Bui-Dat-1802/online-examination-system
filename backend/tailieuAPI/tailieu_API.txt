TÀI LIỆU API BACKEND - BẢN TÓM TẮT

Tài liệu chi tiết, cập nhật theo code hiện tại nằm tại:
backend/tailieuAPI/tailieuAPI.md

Base URL local:
http://localhost:3000/api

Production:
Chưa có domain thật trong repo. Khi triển khai dùng dạng https://<your-production-domain>/api.

Nhóm file theo vai trò:
- backend/tailieuAPI/tailieuAPI_user_chung.md: Auth, user chung, upload ảnh, media import.
- backend/tailieuAPI/tailieuAPI_admin.md: API quản trị viên.
- backend/tailieuAPI/tailieuAPI_teacher.md: API giáo viên, lớp học, câu hỏi, mẫu đề, ca thi, giám sát, import/export.
- backend/tailieuAPI/tailieuAPI_student.md: API sinh viên, lớp học, bài thi, phiên thi, bài làm.
- backend/tailieuAPI/ARCHIVED_DATA_GUIDE.md: Ghi chú dữ liệu xóa mềm/archived theo route hiện tại.

API cũ/không còn sử dụng đã được rà soát:
- POST /api/auth/register: không còn route active.
- /api/exam-import/*: mount trong server.js đang bị comment.
- PUT /api/admin/classes/:id/restore: có controller/service nhưng chưa có route.
- PUT /api/teacher/classes/:id/restore: có controller/service nhưng chưa có route.
- GET /api/teacher/questions?includeDeleted=true: service có hỗ trợ nhưng controller chưa đọc query.
